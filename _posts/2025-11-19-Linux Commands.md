Linux commands notes:
- **ls**: List files and directories  
- **pwd**: Display current working directory  
- **cd**: Change directory  
- **clear**: Clear terminal screen  
- **touch <filename>**: Create a new file  
- **cat <filename>**: View file content  
- **echo "content" > file**: Write content to a file (overwrite)  
- **echo "content" >> file**: Append content to a file  
- **rm <filename>**: Delete a file  
- **mkdir <folder_name>**: Create a new directory  
- **touch "My file"**: Creates two files named My and file (without quotes)  
- **cd..**: Navigate to the parent directory  
- **rm -r <folder_name>**: Recursively delete a directory and its contents  
- **mousepad <filename>**: Open a file in the Mousepad text editor
